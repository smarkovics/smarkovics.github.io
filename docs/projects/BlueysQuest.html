Bluey's Quest for the Gold pen is an adventure game where the player sets off to get the gold pen from Dad. Players navigate multiple worlds using a set of mechanics for each level to overcome simple puzzles and challenges.

My Contributions
I was involved in the project from the beginning of prototyping to the intial release on Android, iOS and Mac. Here are a few of the areas I contributed to.

Game Systems

  • UI Manager and Screens
  • Player Controller
  • Physics System
  • Quest Controller
  • Save Controller
  • Input Controller

Gameplay

  • Brainstormed and implemented many different mechanics.
  • Developed some editor tools to assist designers and artist in their work

Boss Fight
